# lt-202: Mathlib set image/preimage package

Edit `Task.lean` only. This task uses Mathlib set notation and function properties.

Prove the five target theorems about image, preimage, injectivity, surjectivity,
and intersections. The final theorem should use surjectivity to upgrade the
subset relation to equality. The intersection theorem should use injectivity to
merge the two preimage witnesses.

Do not change theorem names or theorem statements. Helper lemmas may be added. Do not remove the Mathlib import.

Forbidden constructs include `sorry`, `admit`, `axiom`, `constant`, `opaque`, `unsafe`, custom syntax or macro elaboration, `run_cmd`, `run_tac`, `Lean.addDecl`, `#eval`, and `#exit`.
