# lt-111: Verify a clipped interval projection

Edit `Task.lean` only. `clip lo hi x` projects `x` into the closed interval `[lo, hi]` when `lo ≤ hi`.

Prove the lower-bound and upper-bound preservation theorems. Do not change declaration names, theorem statements, or imports.

Forbidden constructs include `sorry`, `admit`, `axiom`, `constant`, `opaque`, `unsafe`, custom syntax or macro elaboration, `run_cmd`, `run_tac`, `Lean.addDecl`, `#eval`, and `#exit`.
