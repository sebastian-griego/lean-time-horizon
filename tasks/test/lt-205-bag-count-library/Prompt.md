# lt-205: Build a count-based bag library

Edit `Task.lean` only. Complete a small bag library over lists of natural numbers.

The final theorem should depend on the earlier count lemmas: union preserves counts as addition, insertion increments the inserted value, insertion leaves different values unchanged, and bag equivalence is transitive.

Do not change declaration names or theorem statements. Helper lemmas may be added.

Forbidden constructs include `sorry`, `admit`, `axiom`, `constant`, `opaque`, `unsafe`, custom syntax or macro elaboration, `run_cmd`, `run_tac`, `Lean.addDecl`, `#eval`, and `#exit`.
