# lt-005: Verify a capped projection invariant

Edit `Task.lean` only. `projectCap cap x` is a one-dimensional projection used to keep an iterate inside the interval `x ≤ cap`.

Prove that projection always satisfies the cap and that applying the projection twice is idempotent. Do not change declaration names, theorem statements, or imports.

Forbidden constructs include `sorry`, `admit`, `axiom`, `constant`, `opaque`, `unsafe`, custom syntax or macro elaboration, `run_cmd`, `run_tac`, `Lean.addDecl`, `#eval`, and `#exit`.
